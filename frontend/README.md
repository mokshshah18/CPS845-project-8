# Campus Navigator (Flask + MSSQL + React)

Simple web app that gives walking directions between buildings on a campus map.

---

## Backend Setup

### Prerequisites
- Python 3.10+
- SQL Server running locally or remotely
- ODBC Driver 17 for SQL Server installed

### Install
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
